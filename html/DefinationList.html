Different Parts of web development

HTML

HTML is the standard markup language for Web pages.

With HTML you can create the basic structure of your own Website.

CSS
CSS is the language we use to style an HTML document. CSS describes how HTML elements should be displayed.
Java Script
JavaScript is the programming language of the Web. it is used to add logic and functionality to the webpage
Backend
Backend is the server side program that is responsible for storing and organizing data, and ensuring everything on the client-side works.
